# NTP Export

실제 수출 업무 흐름에 맞춰 주문 접수부터 실출고 입력, 박스 패킹, 국내배송, 엑셀 출력, 사진 증빙까지 관리하는 Streamlit 앱입니다.

## 업무 흐름

1. **오버뷰**
   - 현재 진행 중인 수출 건을 국가, 바이어, 진행 단계와 함께 확인
   - 각 건을 펼치면 주문 제품과 수량 확인
2. **수출 주문 입력**
   - 수출번호 자동 부여
   - 국가, 바이어(선택), 예상출고일, AIR/SEA/HAND, 진행 단계 입력
   - 제품명·수량·단위 주문 목록 저장
3. **실출고 입력**
   - 국가와 진행 중 수출 건 선택
   - 주문 목록 확인
   - WMS에서 `사업장 / 로케이션 / 제품명 / LOT / 유통기한 / 요청수량`을 표에 그대로 붙여넣기
4. **박스 패킹**
   - 실출고 행을 체크한 뒤 원하는 박스번호로 패킹
   - 박스별 가로·세로·높이(cm), 무게(kg) 입력
5. **패킹 결과·배송·엑셀**
   - 국가, 바이어, 운송방식 자동 표시
   - 로젠택배 또는 퀵배송 선택
   - 송장번호 또는 기사 이름·연락처 입력
   - 박스번호와 박스 규격·무게가 병합된 형태로 엑셀 출력
6. **출고 사진**
   - 수출 건별 출고 증빙 사진 업로드 및 조회

## 실행 방법

### Windows에서 바로 실행

`run_app.bat` 파일을 더블클릭하면 됩니다.

처음 실행할 때 자동으로 가상환경을 만들고 필요한 패키지를 설치한 뒤 `app.py`를 실행합니다.

### 직접 실행

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

처음 실행하면 `export.db` SQLite 데이터베이스와 `uploads/` 폴더가 자동 생성됩니다.

## 주요 데이터

- `export_cases`: 수출 건, 국가, 바이어, 운송 및 국내배송 정보
- `order_items`: 주문받은 제품과 수량
- `shipment_items`: WMS에서 가져온 실제 출고 제품
- `boxes`: 박스별 규격과 무게
- `attachments`: 수출 건별 출고 증빙 사진
- `history`: 변경 이력
